# NEAR Generative NFT Landing Page

A minting landing page for generative NFT collections
on NEAR Protocol.

[🚀 Live Demo](https://nearkat-nft.netlify.app)

## Features
- Wallet connection via NEAR Wallet
- Generative NFT minting flow
- Collection gallery view
- Responsive design

## Tech Stack
- React
- NEAR SDK (near-api-js)
- JavaScript
- Netlify deployment

## Getting Started
git clone https://github.com/inc4/near-generative-nft-landing-page.git
cd near-generative-nft-landing-page
yarn install
yarn start

Open http://localhost:3000
